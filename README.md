## API Reference

Link : https://openweathermap.org/current

## Get Weather of city  

```
https://api.openweathermap.org/data/2.5/weather?q={city name}&appid={API key}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**.(Your API key) |
